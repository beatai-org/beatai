# 前言

—— N AVA L R AVI KA N T

你手中握着的，正是创业者唯一需要的书。一位孤独的创始人踏上了一段独特的旅程，去发现并将真理浓缩进一款产品。肩负着责任和期望，这位创始人寻找导师和书籍——但最好的导师无法雇佣，书籍要么是轶 事集要么是学术论著。唯一可靠的老师是苦涩的经历。

在这本书中，埃里克放下了自我，不辞辛劳地以埃隆自己的语言，尽可能务实地汇编了埃隆最精华的经验教训。这不是一本乏 味的传记或事件记述，而是解释与指南。一位热切的联合创始人拿走了我的第一本副本，我的同事们将得到接下来的二十本。

埃隆是自史蒂夫·乔布斯以来最伟大的创业者，但他并非简单的复制。历史不会重演，但会押韵，偶尔出现的拿破仑式人物——无论是在科学、艺术还是政治领域——会用独特的视野和驱动力重新引导历史的必然力 量。勤奋很常见，智慧不常见，勇气则罕见——而埃隆三者兼备，且熊熊燃烧。

平庸的观察会说他是特斯拉、SpaceX、OpenAI 和 Neuralink 的创始人。然而，这些公司只是通往产品的 途径——那些自动驾驶汽车、自主机器人、会思考的机器，以及那些震撼人心、撼动大地、轰鸣升空并脚尖着陆的重型火箭。甚至产品也只是通往使命的途径。只要物理定律允许，并能带我们进入科幻梦想，埃隆就会把我们拖向那里。

财富、名声与嫉妒皆是副产品。困在重力井中的螃蟹们，被政治蒙蔽双眼，会指着他父亲的资金和政府的货币说事。但嫉妒更多地暴露了嫉妒者自身，而非被嫉妒的对象。埃隆睡在工厂地板，每次挫折都加倍投入，永不放弃。值得打动的人们被打动了，数百万孩童受到激励去创造真正的改变。文化源自技术之下游， 技术使我们能够慷慨而自由，埃隆将长久被铭记，而那些夸夸其谈者与碌碌无为者终将被遗忘。

作为首席工程师，埃隆提醒我们，创造价值就是建造事物，而非为其融资或管理。当最富有的人是一个创造财富而非仅仅是转移财富，或打着人民的名义攫取财富的人时，世界会变得更好。

正如物理学家大卫·多伊奇所写，财富是我们能够实现的物理转化的集合——这对个人和社会皆然。财富的主要构成是知识，而非资本。通过创造新知，并将其体现在可复制和分发的人工智能产品中，埃隆和他的同行企业家们成为了财富创造与分配的引擎。

你可能会有机会扪心自问：当人类迈向恒星时，你在做什么？当人类踏上火星时，你是嘲弄他们还是鼓励他们？当他们开采首批小行星并建造第一批空间站时？当自动驾驶汽车车队从停车场和交通拥堵中解放城市时？当瘫痪者重新站立并与我们并肩行走时？当机器人接管了那些艰苦重复的工作时？你是否站在前排欢呼？还是坐在看台上一脸酸楚地讥讽？

甚至还有第三种选择——无论年轻或年老，你的生命尚未完结。埃隆的方法是可复制的，无论大事小事。埃里克已尽可能清晰地向你阐明。如果你的动机纯粹且超越自我，世界将以微妙的方式协力助你。当一人展示了可能性，千万人将涌现，并非作为追随者，而是作为独立的传道者。

你不必效仿埃隆所为，也不必遵循他的方式，但这短暂的生命最值得投入竞技场，投身于超越平凡与永不满足的 自我之事。拒绝渴求安逸与社会认可。重拾青春的乐观精神。远离那些空谈与分裂，踏上学习与建设的道路。

你的能量最好用在与你志同道合的人身上，他们在创造美好事物的使命上势不可挡。不要为了赚货币而造物，要赚货币以便你能造物。不要为工作而获得报酬，要获得报酬以便你能做最好的工作。崛起吧，选择灵感而非嫉妒， 并向你年轻的自己和年长的自己证明，此生所有的渴望都已完完全全、彻彻底底地融入宇宙意志之中。
