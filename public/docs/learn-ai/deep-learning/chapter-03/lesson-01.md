# 函数的极限

## 3.1 函数的极限

### 3.1.1 宇宙是无穷大的

我第一次得知宇宙是无穷大，是在我初中的一堂课上。老师告诉我们说：“什么是无穷大？无穷大就是不论你认为宇宙多大，它比你想的还要大！” 这句话对当时的我造成了很大的震撼。一方面是宇宙竟然没有边。另一方面是老师这种对无穷大的奇妙表达方式。以至于到现在我还清楚的记得这句话。
